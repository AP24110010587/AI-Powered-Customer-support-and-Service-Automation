📖 Project Overview

The AI-Powered Customer Support & Service Automation project is a Salesforce-based CRM solution developed to automate customer service operations using Salesforce's declarative tools and AI capabilities.

The application manages customers, products, orders, and support cases while reducing manual effort through Flow Builder, Validation Rules, Queues, Lightning App Builder, and Prompt Builder.

This project was developed as part of the Salesforce Certified Administrator with AI Agentforce Specialization (SmartBridge) program.
